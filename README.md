# Minesweeper
 Another legacy game recreated in Python using **Pygame**.
 
 Features:
 
 - Menu functionality
 
 - Ability to change the grid size as well as the mine count

 - All the rules that are in the original minesweeper
 
 - Floodfill algorithm for opening cells
